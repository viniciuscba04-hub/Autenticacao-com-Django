# Sistema de Autenticação Django

Sistema de autenticação backend em Python/Django com registro, login e logout de usuários, sem APIs externas.

Funcionalidades:

- Registro e login de usuários

- Logout seguro

- Senhas armazenadas com hash seguro

- Rotas protegidas para usuários autenticados

Tecnologias:

- Python 

- Django 

- SQLite
